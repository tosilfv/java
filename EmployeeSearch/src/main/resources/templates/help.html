HELP

Alias

Change Alias

Alias cannot be changed after registration. If you want a new alias, you could try to create a new profile with a new alias if it is not already taken.

Password

Forgotten password

If you forget your password, there is no way of recovering it through this Site. The Site does not provide this service.

Profile

Delete Profile Picture

Deleting your Profile Picture means that you cannot recover your previous Profile Picture anymore, with no exceptions. To delete the current Profile Picture, stock or loaded, just select Neutral from the Select Stock Profile Picture: dropdown Menu on the Preferences Page and click the corresponding SELECT button. This action will delete your previous loaded Profile Picture from the Site's database.

Delete Profile

Deleting your Profile means that you cannot log in to your Profile anymore and that your Profile and all information that was in it on this Site is removed from the Site's database. If you wish to learn more about the Site's privacy policy, see terms of service - Privacy policy.

Hide Profile

Hiding your Profile means that no one else can see your Profile in this Site except you. This action does not hide any posts, comments, likes etc. that you have made. It hides only your Home Page and all the contents that are in it.
